# Progression monitor
This is a record of knowledge and skills that I covered and got familiar with, as well as feeling confident using them without any support.
The .py files in this repository are as evidence. While they are nothing spectacular, they showcase the basic fundamentals.
A single basic skill may not create a fully functional program, but a collection of them is.
Mandatory basic knowledge that every developer should know is not being covered such as what is a variable or what is an operator.
